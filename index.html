Sobre o HTML

(Voltar)

É uma Linguagem de Marcação de Hipertexto (HyperText Markup Language). Utilizada na criação de páginas web, é composta por tags, cada uma com seu propósito. A versão dessa linguagem mais utilizada ultimamente é o HTML5, mais prático e simples de entender que suas versões anteriores. Sua sintaxe é baseada no uso de < e >. Através dele, podemos criar títulos, parágrafos, inserir imagens, etc, tendo diversas possibilidades. Este é um exemplo de página criada a partir de código de hipertexto. O que são tags?

(Voltar)

São elementos usados para marcar e formatar o conteúdo HTML. Cada uma possui seu nome e sintaxe. A maioria possui tag de fechamento, que é de mesmo nome com uma / (barra) na frente.

Por exemplo: <p>Criação de um parágrafo</p>


Principais tags

(Voltar)

As tags básicas, portanto, as mais utilizadas são:

  • h1-h6: utilizadas para títulos. Seu número indica seu nível de destaque. O h1, por exemplo, é usado para título principal.
  • p: utilizada para indicar um parágrafo.
  • mark: grifa um trecho de texto. Exemplo: texto.
  • small: deixa um trecho pequeno. Exemplo: texto.
  • i: deixa um trecho em itálico. Exemplo: texto.
  • u: deixa um trecho sublinhado. Exemplo: texto.
  • strong: deixa um trecho em negrito. Exemplo: texto.
  • ol, ul e li: utilizadas para listas. Enquanto ol representa uma lista ordenada, ul representa o contrário. Já li é utilizada dentro de uma das tags anteriores, pois indica cada item da lista.
  • a: vem de anchor. Cria um hyperlink que redireciona o usuário para uma página externa ou um conteúdo específico da própria página, assim como o botão (Voltar) próximo aos subtítulos.
  • hr: cria uma linha de separação.
  • sup: transforma o texto em uma "potência". Exemplo: texto.
  • sub: contrário do sup. Ao invés do texto ficar em cima, fica em baixo. Exemplo: texto.
  • blockquote: cria um bloco de citação.
